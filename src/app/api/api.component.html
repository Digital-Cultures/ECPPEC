ECPPEC Data

Our data is downloadable through the map but is also available through our API. You can query the API without login credentials but if you want to contribute data, you'll need an API key. You can request one here.

Full API documentation is coming but for now our API end points are here:


https://ecppec.ncl.ac.uk/php/getElections.php
https://ecppec.ncl.ac.uk/php/getPollBooks.php

You them add query terms to them like this:


https://ecppec.ncl.ac.uk/php/getElections.php/?from_year=1700&to_year=1800
https://ecppec.ncl.ac.uk/php/getElections.php/?from_year=1700&to_year=1800&month=July
https://ecppec.ncl.ac.uk/php/getElections.php/?from_year=1700&to_year=1800&month=July;June;Sept
